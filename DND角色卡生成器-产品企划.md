# DND 角色卡生成器 — 产品企划书

> AI 驱动的 D&D 5e 角色创作平台 | 开源项目

---

## 一、项目概述

### 1.1 产品愿景

打造一个 **AI 驱动的 D&D 5e 角色创作平台**，让玩家不仅能快速创建规则合规的角色卡，还能通过 AI 生成丰富的角色背景故事、人格特质和角色立绘，带来沉浸式的角色创作体验。

### 1.2 核心差异化

| 维度 | D&D Beyond | Friends & Fables | **我们的产品** |
|------|-----------|-----------------|---------------|
| 角色创建 | 引导式表单 | AI 对话式 | **AI 引导 + 可视化编辑** |
| 背景故事 | 手动填写 | AI 生成（内嵌在游戏中） | **独立的 AI 故事工坊** |
| 角色立绘 | 无 | AI 生成 | **AI 生成 + 风格自选** |
| 规则引擎 | 完整付费内容 | 简化版 | **基于 SRD 5.2 开源数据** |
| 定价 | 订阅制 | 订阅制 | **完全开源免费** |
| 平台 | Web | Discord Bot | **Web（React/Next.js）** |

### 1.3 目标用户

- **新手玩家**：第一次接触 D&D，需要引导式体验
- **跑团老手**：快速创建和管理多个角色
- **DM（地下城主）**：批量生成 NPC、管理团队角色
- **TRPG 创作者**：需要丰富角色灵感和素材

### 2.4 叙事学、神话、宗教与哲学——AI 文本的深层骨架

仅靠游戏 Lore 和 SRD 数据，AI 生成的文本只能做到"设定正确"。要实现真正有深度、有文学性的输出，我们需要将**叙事学理论、世界神话原型、宗教哲学思想**作为 AI 的深层骨架注入。

#### 2.4.1 叙事学理论框架

以下三大叙事理论构成我们 AI 故事生成的"结构引擎"：

**A. 坎贝尔《千面英雄》— 英雄之旅 (Hero's Journey)**

坎贝尔从全球神话中提炼出的"单一神话"模型（1949），是理解绝大多数冒险叙事的底层框架：

```
  日常世界 → 冒险召唤 → 拒绝召唤 → 遇见导师 → 跨越阈限
      ↓                                              ↓
  携万灵药归来 ← 复活 ← 回程 ← 获得报偿 ← 考验/盟友/敌人
```

Vogler 将坎贝尔的 17 阶段简化为 12 阶段，并定义了**八种角色原型**：英雄(Hero)、导师(Mentor)、阈限守卫(Threshold Guardian)、信使(Herald)、变形者(Shapeshifter)、阴影(Shadow)、盟友(Ally)、骗术师(Trickster)。

**我们的应用**：AI 生成背景故事时，自动分析角色处于英雄之旅的哪个阶段。一个 1 级角色可能刚经历"冒险召唤"或"跨越阈限"；一个高等级角色可能正处于"深渊考验"或"回程"阶段。同时，AI 为角色的 NPC 关系网中的每个人物分配一个**原型角色**。

**B. 普罗普《故事形态学》— 31 功能单元**

俄国民俗学家普罗普从俄罗斯民间故事中提取了 31 个叙事功能和 7 类角色功能。核心洞见：**所有故事都由有限的叙事单元排列组合而成**。

**我们的应用**：将普罗普的功能单元作为"叙事积木"——AI 从中抽取 3-5 个功能单元，组合成独特的背景故事骨架。例如：
- 功能8（加害）+ 功能11（出发）+ 功能14（获得魔法物品）+ 功能25（考验）
- → "你的家族被一次神秘的灾变摧毁（加害）。你离开废墟（出发），在路上发现了一件遗物（魔法物品）。这件遗物的代价至今在你的梦中索取（考验）。"

**C. 布克《七种基本故事》— 七大叙事原型**

Christopher Booker 用 34 年写就的理论，识别出人类故事的七种元结构：

| 原型 | 核心弧线 | D&D 角色适用场景 |
|------|---------|----------------|
| **屠龙** (Overcoming the Monster) | 直面强大的邪恶力量并战胜它 | 圣骑士/战士的经典弧线 |
| **白手起家** (Rags to Riches) | 从卑微到伟大的崛起 | 孤儿/街头混混出身的角色 |
| **远征** (The Quest) | 踏上漫长旅途寻找某物 | 游侠/朝圣者类型 |
| **远航与归来** (Voyage and Return) | 进入异世界，经历变化后归来 | 穿越位面、接触异域文明 |
| **喜剧** (Comedy) | 误解与混乱最终走向和解 | 吟游诗人/骗术师角色弧线 |
| **悲剧** (Tragedy) | 角色因自身缺陷走向毁灭 | 堕落圣骑士/术士的献身之路 |
| **重生** (Rebirth) | 被囚于黑暗状态后获得救赎 | 前罪犯/被诅咒者的救赎之旅 |

**我们的应用**：在角色创建时，AI 根据用户选择的种族/职业/背景/缺陷，**自动推荐最匹配的叙事原型**，并据此生成背景故事的整体弧线。用户也可以手动选择或组合多种原型。

#### 2.4.2 荣格原型心理学——角色内心的深层结构

荣格（Carl Jung）的原型理论为角色心理提供了比"人格特质/理想/羁绊/缺陷"更深的框架：

**四大核心原型：**

| 原型 | 含义 | 在角色卡中的体现 |
|------|------|----------------|
| **人格面具 (Persona)** | 角色展示给世界的"面具" | 角色对外表现出的身份——"受人尊敬的牧师"、"冷酷的雇佣兵" |
| **阴影 (Shadow)** | 被压抑的、不愿面对的自我 | 角色的"缺陷"深层版本——不只是"贪财"，而是"你害怕成为你父亲的样子，但你已经在镜子里看到了他的脸" |
| **阿尼玛/阿尼姆斯 (Anima/Animus)** | 内在的异性原型，代表未整合的灵魂面 | 角色内心深处渴望成为但害怕承认的另一面——一个暴力战士心中的诗意渴望 |
| **自性 (Self)** | 人格整合的最终目标 | 角色冒险的终极心理目标——不是"打败魔王"，而是"成为完整的自己" |

**十二种荣格人格原型（Character Archetypes）：**

| 原型 | 核心驱力 | 恐惧 | D&D 角色映射 |
|------|---------|------|-------------|
| 天真者 (Innocent) | 追求幸福 | 被惩罚 | 新手冒险者、理想主义牧师 |
| 探索者 (Explorer) | 追求自由 | 被困住 | 游侠、浪迹天涯的吟游诗人 |
| 智者 (Sage) | 追求真理 | 无知/被骗 | 法师、学者、调查者 |
| 英雄 (Hero) | 追求掌控 | 软弱/脆弱 | 战士、圣骑士、经典冒险者 |
| 叛逆者 (Outlaw) | 追求解放 | 被压迫 | 游荡者、反建制术士 |
| 魔法师 (Magician) | 追求力量 | 意外后果 | 术士、邪术师、炼金术士 |
| 照顾者 (Caregiver) | 追求服务 | 自私/被忽视 | 牧师、德鲁伊、治疗者 |
| 统治者 (Ruler) | 追求控制 | 混乱/被推翻 | 贵族背景、战术领袖 |
| 创造者 (Creator) | 追求创新 | 平庸 | 工匠、铸造师、发明者 |
| 凡人 (Everyman) | 追求归属 | 被排斥 | 平民出身的冒险者 |
| 弄臣 (Jester) | 追求欢乐 | 无聊/无趣 | 吟游诗人、骗术师 |
| 情人 (Lover) | 追求亲密 | 孤独/不被爱 | 魅力系角色、外交官 |

**我们的应用**：
- 角色创建时，AI 在幕后自动分析用户选择的种族/职业/背景/人格组合，推断出**主导原型（1个）+ 次要原型（1-2个）+ 阴影原型（1个）**
- 阴影原型是主导原型的"暗面"——英雄的阴影是暴君，照顾者的阴影是殉道者，智者的阴影是冷血旁观者
- 这些原型**不直接暴露给用户**，而是作为 AI 文本生成的深层指导——确保角色的行为模式、内心冲突和成长弧线心理上自洽

#### 2.4.3 世界宗教与信仰体系——超越"西方中世纪"的世界观选择

D&D 的默认设定偏重西方中世纪基督教色彩的善恶二元论。为了提供更丰富、更深刻的世界观选项，我们从全球信仰体系中提取叙事灵感：

**A. 印度教 (Hinduism) — 循环宇宙论与神的化身**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **三相神 (Trimurti)** | 梵天（创造）/ 毗湿奴（维护）/ 湿婆（毁灭）——不是善恶对立，而是宇宙的三个必要面向 | 信仰系统设计：神不必是"善/恶"，可以是"创造/维持/必要的毁灭"。一个侍奉毁灭之神的牧师不一定是邪恶的——毁灭是新生的前提 |
| **化身 (Avatara)** | 毗湿奴的十大化身——神以不同形态降临凡间，每次应对不同的宇宙危机 | 角色可能是某个神的"碎片化身"而不自知。或者角色的血脉中混入了某次神之降临的余波 |
| **达摩 (Dharma)** | 每个人有独特的使命与正道——不是普世的道德法则，而是属于你个人的道 | 比 D&D 阵营系统更细腻的道德框架：角色的"善"不是抽象的，而是"你特定角色的正确道路" |
| **业力 (Karma)** | 因果循环——不是简单的"做好事得好报"，而是跨越轮回的因果网络 | AI 生成背景故事时，可以暗示角色的某些际遇是"前世业力的回响"——一种不可解释的恐惧、一种莫名的亲切感 |
| **摩耶 (Maya)** | 世界是幻象——真实隐藏在表象之后 | "幻象"主题：角色的记忆、感知甚至整个世界可能不是它看起来的样子 |

**B. 佛教 (Buddhism) — 苦、空、觉醒**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **四圣谛** | 苦（Dukkha）→ 集（苦因）→ 灭（苦灭）→ 道（通往灭苦之路） | 角色弧线的替代模板：不是"打败敌人"，而是"理解痛苦的本质并找到超越之路" |
| **六道轮回** | 天道/修罗道/人道/畜生道/饿鬼道/地狱道——不同的存在状态，由业力决定 | D&D 已有位面系统，但佛教六道提供了更深的心理隐喻——角色可能在心理上"堕入饿鬼道"（永不满足的贪婪）或"修罗道"（永无止境的争斗） |
| **菩萨道** | 已经可以涅槃但选择留在轮回中度化众生 | 一种强大的角色动机模板："你本可以离开，但你选择留下。不是因为执念，而是因为慈悲。" |
| **缘起 (Pratītyasamutpāda)** | 万物因缘和合而生，无一独立存在 | 角色的故事不是孤立的——每个选择都与他人的命运相互依存。AI 生成时强调关系网络而非孤胆英雄 |
| **空 (Śūnyatā)** | 一切现象皆无固有本质 | 最深层的角色哲学：身份本身是流动的。"你是谁？"这个问题可能没有固定答案——这是角色的终极探索 |

**C. 琐罗亚斯德教 (Zoroastrianism) — 光暗之战与自由意志**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **阿胡拉·马兹达 vs 安格拉·曼纽** | 善神与恶神的宇宙战争——但不是无尽循环，而是有终局（9000年之战） | 与 D&D 传统善恶轴线最接近，但更精致：宇宙是一个**有限时间的战场**，每个凡人的选择都在加速或延缓终局 |
| **阿沙 (Asha) vs 德鲁杰 (Druj)** | 真理/秩序 vs 谎言/混乱 | 不是"善 vs 恶"，而是**"真实 vs 虚伪"**——圣骑士的使命不是杀恶人，而是揭穿谎言、恢复真实 |
| **自由意志** | 人类不是被动的棋子，而是宇宙战争中有自由选择权的能动者 | 强调角色的**选择**——你的善行不是因为神命令你，而是因为你在全然自由的状态下做出了选择 |

**D. 神道教 (Shinto) — 万物有灵与净秽**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **神 (Kami)** | 万物皆有灵——山川、河流、古树、风雨、甚至刀剑和房屋都可能有自己的"神" | 装备和环境描述的深度维度：一把用了百年的剑不只是有魔法附魔，它有自己的"意志"和"记忆" |
| **秽 (Kegare) 与 祓 (Harae)** | 污秽不是道德上的"罪"，而是一种需要被净化的状态。死亡、疾病、破坏都会带来秽 | 替代"善恶"的框架——角色可能不是"邪恶"的，而是"被污秽了的"，需要的是净化而非惩罚 |
| **三层宇宙** | 高天原（神域）/ 苇原中国（人间）/ 黄泉国（死域） | 简洁但深邃的宇宙结构。死后世界不是地狱，而是一个"所有事物最终回归的寂静之地" |

**E. 约鲁巴信仰 (Yoruba) — 中介者与命运**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **奥利沙 (Orisha)** | 至高神 Olodumare 的中介者——数百位神灵各掌管生活的不同面向（雷电、河流、铁器、十字路口） | 比 D&D 的领域系统更丰富的神祇设计——不只是"战争之神"，而是"十字路口之灵，守护每一个选择的分岔" |
| **奥里 (Ori)** | 每个人的"内在头颅"——个人命运和精神本质 | 角色的 Ori 是他们在出生前选择的命运。但——他们可能选错了。或者忘记了自己选择的是什么 |
| **伊法 (Ifá)** | 神谕系统——通过占卜揭示宇宙的模式和个人的道路 | AI 可以生成"神谕式"文本——不给明确答案，而是给出需要解读的象征和隐喻 |

**F. 中美洲信仰 (Mesoamerican) — 九层冥界与血的契约**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **米克特兰 (Mictlan)** | 阿兹特克的九层冥界——死后灵魂必须经过九层考验才能安息。不是惩罚，而是最后的旅程 | "死亡"在角色背景中不只是"家人死了"——它可以是一次九层净化之旅的起点。角色可能曾"穿越冥界并回来"，但留下了一部分自己 |
| **西巴尔巴 (Xibalba)** | 玛雅冥界，有六座致命之屋——黑暗之屋、寒冷之屋、美洲豹之屋、蝙蝠之屋、刀刃之屋 | 地下城设计灵感：不是随机的陷阱走廊，而是有主题、有考验意义的空间 |
| **血的契约** | 众神用自己的血创造了人类，因此人类有义务回馈——以仪式、牺牲和行动 | 一种不同于"信仰=祈祷"的宗教关系——你的神给了你生命，**你欠他们血** |

**G. 澳大利亚原住民 (Aboriginal) — 梦境时间与歌之路**

| 概念 | 内涵 | AI 叙事应用 |
|------|------|------------|
| **梦境时间 (Dreamtime)** | 不是"过去"，而是一个永恒存在的创世维度——祖先精灵通过行走、歌唱和做梦来创造世界，而且他们**仍在创造** | 时间不必是线性的。角色的背景故事可以包含"在梦境时间中发生的事"——不是回忆，而是一个永远正在发生的平行叙事 |
| **歌之路 (Songlines)** | 祖先走过的路径就是歌曲，歌曲就是地图。唱对了歌，就能穿越世界 | 吟游诗人/德鲁伊的终极灵感——"你学会了一首歌。这首歌比你古老。当你唱它时，你走过的路就与创世之路重合" |

#### 2.4.4 哲学流派与角色动机系统

每个角色的深层动机可以映射到一种哲学立场。AI 根据角色的选择和人格，在幕后分配一个**哲学倾向**，指导所有后续文本生成：

| 哲学 | 核心主张 | 角色行为模式 | 典型角色 |
|------|---------|------------|---------|
| **斯多葛主义** | 专注于可控之事，平静接受不可控之事 | 沉着、克制、义务导向，在混乱中保持内心秩序 | 老练战士、沉默的修士、忠诚的侍卫 |
| **存在主义** | 存在先于本质——你通过行动定义自己 | 追求真实性、拒绝预设角色、在荒诞中寻找意义 | 离经叛道者、质疑传统的牧师、寻找自我的变形者 |
| **荒诞主义** | 生命本无意义，但我们选择反抗 | 明知徒劳仍然行动、黑色幽默、拒绝虚假安慰 | 曾目睹真相仍选择战斗的冒险者、用笑声对抗绝望的吟游诗人 |
| **虚无主义** | 一切价值都是人为建构，没有客观意义 | 冷漠与爆发交替、解构他人的信仰、可能走向毁灭或觉醒 | 失去信仰的牧师、背叛组织的刺客、看透一切的法师 |
| **功利主义** | 最大多数人的最大幸福 | 冷酷的计算、可以为"大善"牺牲少数人、结果导向 | 政治家型角色、战争指挥官、"肮脏双手"式英雄 |
| **康德义务论** | 存在绝对的道德律令，不论后果 | 绝不撒谎、绝不把人当工具、即使世界毁灭也不破例 | 执着的圣骑士、刻板的法师、不妥协的审判者 |
| **道家自然主义** | 道法自然、无为而治——最好的行动是顺应而非强求 | 柔软、适应、看似消极实则精准、水的哲学 | 德鲁伊、武僧、隐士 |
| **犬儒主义** | 拒绝社会规范、追求符合自然的生活 | 挑衅权威、嘲讽虚伪、极简生活、刺耳的诚实 | 街头哲人、反英雄、流浪者 |
| **柏拉图主义** | 存在超越物质的理念世界，真实隐藏在表象之后 | 追寻隐藏的真理、对"完美形式"的执念、对物质世界的不满 | 法师、预言者、理想主义者 |
| **尼采权力意志** | 超越传统道德，成为"超人"——自我立法、自我创造 | 打破规则、强烈的个人意志、蔑视平庸、危险的魅力 | 术士、野心家、革命者 |
| **儒家仁义** | 仁者爱人、以礼制约、修身齐家治国 | 重视关系和等级、义务先于自我、文化教养、家族荣誉 | 贵族战士、师长型角色、守护传统的卫士 |
| **佛教中道** | 避免极端、觉察执念、慈悲为怀 | 不执着于胜负、在对立中寻找第三条路、平静面对无常 | 武僧、治疗者、调停者 |

**我们的应用**：
- AI 在幕后为每个角色分配 1-2 个哲学倾向（**不直接显示给用户**，避免说教感）
- 哲学倾向影响 AI 生成的所有文本——同样的事件，斯多葛角色会"默默承受"，荒诞主义角色会"大笑着面对"
- 用户可以在高级设置中手动选择或调整哲学倾向

#### 2.4.5 意识流文学——记忆碎片的桥接与内心独白的深渊

角色的背景故事不应是线性的"编年史"，而应像**记忆本身**一样运作——跳跃、重叠、由感官触发、被情感扭曲。意识流（Stream of Consciousness）是20世纪文学最重要的革命之一，它揭示了人类思维的真实运作方式：不是逻辑的链条，而是**感觉的河流**。

我们从四位意识流大师中提取核心技法，将其转化为 AI 角色叙事的生成策略：

**四大意识流源泉：**

| 作家 | 核心技法 | 角色叙事应用 |
|------|----------|-------------|
| **普鲁斯特**（Marcel Proust） | **非自主记忆**——一个感官细节（气味/味道/触感）引爆整段被遗忘的过往 | 角色背景中的"触发物"设计：一块玛德莲蛋糕般的物件，串联起童年、创伤、转折 |
| **伍尔夫**（Virginia Woolf） | **存在的瞬间**（Moments of Being）——在日常中突然感知到存在本身的重量 | 角色背景中嵌入"顿悟时刻"：不是通过大事件，而是通过一个平凡瞬间改变一切 |
| **乔伊斯**（James Joyce） | **无标点内心独白**——思维的原始流动，无过滤、无编辑、无逻辑连接 | AI 生成角色内心声音时，模拟真实思维的断裂与跳跃，拒绝完美的叙述 |
| **福克纳**（William Faulkner） | **时间碎片化**——过去与现在交织，时间不再线性，记忆侵入当下 | 角色背景的"时间折叠"：同一段文字中过去和现在并行，不标注时间切换 |

**五大记忆桥接机制：**

这些机制定义了 AI 如何在角色背景的不同记忆片段之间创建自然的过渡，而非使用"后来"、"之后"这类机械连接词：

1. **感官锚点**（Sensory Anchor）—— 普鲁斯特式
   - 一个感官细节同时存在于两段记忆中，成为桥梁
   - 示例：*"铁锈的气味——他低头看着剑上的血迹——不，那是父亲铁匠铺里的气味，他七岁时……"*

2. **情感共振**（Emotional Resonance）—— 伍尔夫式
   - 两段记忆共享同一种情感状态，通过情感的相似性自然切换
   - 示例：*"那种被抛弃的感觉又回来了。像地下城深处的回声，像母亲转身离开时斗篷扬起的风……"*

3. **语词回声**（Verbal Echo）—— 乔伊斯式
   - 一个词或短语在不同语境中重复出现，每次携带不同含义
   - 示例：*"守护。他发誓要守护这座城。守护。母亲也曾这样说过，守护好你的弟弟。他没能。守护。这个词如今只剩下嘲讽的味道。"*

4. **无预告时间跳转**（Unannounced Time Shift）—— 福克纳式
   - 不使用任何时间标记词，直接在过去与现在之间切换
   - 示例：*"她举起法杖对准了巨龙的眼睛。小时候她也这样举着木棍，对准后院的稻草人，祖母在窗户后面笑。巨龙咆哮了。"*

5. **螺旋式逼近**（Spiral Approach）—— 综合技法
   - 同一核心记忆被反复触及，每次揭示更多细节，像螺旋一样逐渐逼近真相
   - 适用于角色的核心创伤或秘密——第一次提及只有模糊的影子，最终才完整呈现

**三级意识流强度控制：**

| 级别 | 名称 | 描述 | 适用场景 |
|------|------|------|----------|
| Lv.1 | **微澜**（Ripple） | 在常规叙事中偶尔插入一两句意识流片段，如记忆闪回或感官联想 | 大多数角色背景，保持可读性 |
| Lv.2 | **暗流**（Undercurrent） | 叙事结构开始松动，时间线偶尔交错，内心独白与外部叙述交织 | 有创伤或复杂心理的角色 |
| Lv.3 | **洪流**（Torrent） | 完全的意识流体验——无标点长句、时间完全碎片化、多重记忆同时涌现 | 实验性角色或特定叙事时刻（如角色濒死时的走马灯） |

**两种新增文学风格：**

**风格 15：记忆之河（River of Memory）**
- 灵感：普鲁斯特 × 伍尔夫
- 特征：绵长的感官描写触发记忆连锁，句式悠长而富有韵律，时间在"现在的感知"与"过去的体验"之间流淌
- Prompt 指令片段：`以一个当下的感官细节（气味/光线/声音）开始，让它自然地唤醒一段被遗忘的记忆。句子应该像河水一样绵长，用逗号而非句号连接思绪。每段记忆都要包含至少一个具体的触觉或嗅觉细节。`

**风格 16：碎片拼图（Shattered Mosaic）**
- 灵感：乔伊斯 × 福克纳
- 特征：极短的句子碎片与无标点长句交替出现，时间完全非线性，同一段落中可能存在三个不同时间点
- Prompt 指令片段：`将角色的记忆打碎成不规则的碎片。短句。断裂。然后突然一段没有标点的长句如洪流般涌出将过去现在未来搅在一起。用"语词回声"技法——让一个关键词在不同段落中重复出现，每次含义都不同。`

#### 2.4.6 影像叙事学——镜头语言与文学语言的三维融合

传统角色背景故事是**一维的文字**。我们的目标是让 AI 生成的文本具备**三维性**——像电影一样同时在"视觉层"、"听觉层"和"心理层"上展开，让读者在阅读文字时自动在脑中"看到"画面。

这一设计灵感来自《爱、死亡和机器人》的短篇叙事密度、塔可夫斯基的诗意影像、王家卫的情绪质感、林奇的梦境逻辑，以及黑泽明的构图哲学。

**A. 核心理论：电影技法到文学技法的转译表**

| 电影语言 | 文学转译 | 效果 | Prompt 指令示例 |
|----------|---------|------|----------------|
| **特写镜头 (Close-up)** | 聚焦单一感官细节——一只手的纹路、瞳孔中的火光、血滴落在石板上的声音 | 制造亲密感和紧张感 | "描写这个瞬间时，将叙事焦距收窄到一个微小的感官细节上" |
| **远景/建立镜头 (Wide/Establishing)** | 先描述宏大的环境——天空的颜色、城市的轮廓、远处战场的烟——再让角色出现 | 制造渺小感和史诗感 | "从这个世界的广袤开始，让角色像一个小点般进入画面" |
| **跟踪长镜头 (Tracking Shot)** | 不断裂的连续散文段落，跟随角色的移动，途经的每个细节都自然呈现 | 制造沉浸感和流动感 | "用一个不断裂的长段落跟随角色穿过这个空间，让环境自然展开" |
| **蒙太奇 (Montage)** | 快速切换的短句/短段，时间跳跃，每个片段只有一个核心意象 | 制造时间压缩感和节奏感 | "用5-7个短句蒙太奇展示这段岁月的流逝，每句一个画面" |
| **升格/慢镜头 (Slow-mo)** | 极度膨胀的时间——一秒钟的动作用整段文字描写 | 制造关键时刻的仪式感 | "这个瞬间在角色的感知中被无限拉长。用整段文字描写这一秒" |
| **景深/焦点转换 (Rack Focus)** | 描述中突然从前景切换到背景，或反之——正在说话的人模糊了，他身后墙上的符文清晰了 | 引导注意力，暗示隐藏信息 | "让叙事焦点从当前主体滑向背景中一个被忽略的细节" |
| **倾斜构图 (Dutch Angle)** | 句法不稳定——句子在不该断的地方断开、主语缺失、时态突变 | 制造不安感和失序感 | "用倾斜的语法写这一段——打断句子结构，制造语言层面的眩晕" |
| **画外音/旁白 (Voice Over)** | 第二人称或第三人称的"超叙事声音"插入，打破第四面墙 | 制造间离感和哲思感 | "在这段叙述中插入一个来自'更高处'的声音，仿佛有人在注视角色的一生" |
| **色彩调色 (Color Grading)** | 用颜色词汇控制段落的情绪基调——琥珀色=温暖回忆，青灰色=疏离，暗红色=暴力暗涌 | 控制阅读情绪 | "这一段的视觉调性是[颜色]的。让所有意象浸泡在这个色调中" |
| **声音设计 (Sound Design)** | 在视觉描写间织入声音——不是"他听到了风声"，而是"风在窗格间发出一种像指甲划过丝绸的声音" | 制造感官立体感 | "每段视觉描写之间必须穿插一个精确的声音细节" |

**B. 影视作品灵感库——叙事氛围模板**

从经典影像作品中提炼"氛围模板"，AI 可以将其作为文本生成的情绪参照：

**《爱、死亡和机器人》— 短篇叙事的极致密度**

| 集名 | 叙事精华 | AI 文本借鉴 |
|------|---------|------------|
| **齐马蓝 (Zima Blue)** | 宇宙级的存在主义寓言压缩在 10 分钟内。一个AI艺术家穿越宇宙寻找终极真理，最终发现答案是他最初的形态——一个清洁泳池的小机器人。极简美学 + 哲学深度 | **"回归本源"叙事模式**：角色的终极答案不在远方，而在起点。AI 生成的最深刻背景故事应该是**回环的**——结尾照见开头 |
| **吉巴罗 (Jibaro)** | 零对话，纯视觉+声音叙事。金甲塞壬与聋骑士的致命舞蹈。贪婪、殖民、毒性关系的多层寓言。"寓言套寓言套寓言" | **"无言叙事"模式**：AI 可以生成几乎没有直接对话的段落——纯粹通过动作、感官和环境来讲述。同时，**一个故事承载多层隐喻** |
| **超越阿奎拉裂缝 (Beyond the Aquila Rift)** | 柏拉图洞穴寓言的科幻版。你以为的现实是被精心编织的幻象，真相是宇宙级的恐怖 | **"认知剥离"模式**：背景故事中的某些"事实"可能是角色为了保护自己而构建的虚假记忆。AI 可以在叙事中埋入**真实性裂缝**——读者（和角色）无法确定哪些记忆是真的 |
| **溺水的巨人 (The Drowned Giant)** | 纯粹的冥想式叙述。一具巨人遗体冲上海滩，人们的反应从敬畏到亵渎。博尔赫斯式的寓言 | **"旁观者冥想"模式**：不通过角色自述，而是通过一个旁观者的冷静叙述来侧面展现角色。适合NPC视角的角色描述 |

**塔可夫斯基 (Tarkovsky) — 时间的雕塑**

| 技法 | 电影实现 | AI 文本转译 |
|------|---------|------------|
| **"诗意静止"** | 《潜行者》中镜头在废墟水面上停留数分钟，水面下可见沉没的物品——注射器、宗教图标、硬币 | AI 生成环境描述时的**"停留"技法**——不急于推进情节，让叙事在一个空间中驻留，通过细节的累积制造诗意密度 |
| **"元素语言"** | 风、水、火、雾不是背景，而是有情感的角色。雨的方式不同，情绪就不同 | AI 描写自然元素时，赋予其**情感主动性**——"雨不是落下来的。雨是从某个更高的悲伤中被释放的" |
| **"废墟美学"** | 自然吞噬人工的景象——铁轨上的青苔、裂缝中的花——是塔可夫斯基最核心的视觉母题 | 角色的记忆/过往场景描写中，**时间的物质痕迹**是关键——"那座塔还在，但树已经从窗户里长了出来" |

**王家卫 (Wong Kar-wai) — 情绪的色彩学**

| 技法 | 电影实现 | AI 文本转译 |
|------|---------|------------|
| **"步调印刷"** | 慢动作中角色清晰，周围世界模糊——时间属于某一个人 | AI 在描写关键情感瞬间时，**让世界模糊、让角色锐利**——"周围的一切都在加速流逝。只有她转身的那个动作，慢得像是水中的动作" |
| **"缺席的对话"** | 《花样年华》中最重要的话从未被说出。情感通过走廊中的擦肩、共用一面墙的沉默传达 | AI 生成角色关系时，**未说出的话比说出的话更有力**——"你们之间最重要的对话发生在沉默中。你知道。他也知道" |
| **"色彩即情绪"** | 红色走廊=欲望与压抑，蓝绿霓虹=都市孤独，暖黄=流逝的记忆 | AI 文本的**段落调色**系统——每段主导一种色彩情绪，不直说情绪，而是让颜色"浸透"所有意象 |

**大卫·林奇 (David Lynch) — 梦的语法**

| 技法 | 电影实现 | AI 文本转译 |
|------|---------|------------|
| **"梦境逻辑"** | 因果关系被打破，场景之间的过渡不遵循物理逻辑——一个房间的门打开后是另一个时代 | AI 在描述角色的梦境、幻觉或创伤记忆时，使用**断裂因果**——"你推开门。门后不是走廊。门后是你六岁时的厨房。桌上的面包还是温的" |
| **"诡异谷"** | 日常场景中注入一个微妙的"不对"——正常的餐厅，但所有人都在微笑，而且是同一个微笑 | AI 在描述看似正常的场景时埋入**微妙的不协调**——制造一种说不出的不安 |
| **"双重身份"** | 同一个角色在不同叙事层中是不同的人——但共享某些细节（一首歌、一个手势、一条蓝色丝绒带） | AI 在探索角色的多面性时，用**重复的微小细节**连接角色的不同面向——同一个手势在不同记忆中出现，但含义完全不同 |

**黑泽明 (Kurosawa) — 构图即叙事**

| 技法 | 电影实现 | AI 文本转译 |
|------|---------|------------|
| **"天气即灵魂"** | 大雨=内心冲突的外化，阳光突破乌云=领悟的瞬间 | AI 让天气成为角色内心状态的**外在回声**——不是"他很悲伤，外面在下雨"，而是"雨先开始的。然后他才意识到自己在悲伤" |
| **"对角构图"** | 两个对立力量被放置在画面的对角——制造张力 | 在描写角色与对立面的相遇时，用**空间对位**——"他站在房间的这一端。她站在那一端。中间那条光线像一道无法跨越的河" |
| **"运动即性格"** | 《七武士》中每个武士的移动方式都不同——性格通过肢体语言展现 | AI 描写角色动作时，**动作方式即性格素描**——沉稳的人"将杯子放下时没有发出声音"，暴烈的人"走路时地板知道他来了" |

**C. "三维叙事"AI 生成框架**

将上述理论整合为 AI 文本生成的**三层同时渲染框架**：

```
┌─────────────────────────────────────────────────────────┐
│                 三维叙事渲染引擎                           │
│                                                           │
│  第一维：视觉层 (What You See)                            │
│  ├─ 镜头类型选择（特写/远景/跟踪/蒙太奇）                 │
│  ├─ 构图方式（对称=秩序/不对称=冲突/空旷=孤独）           │
│  ├─ 色彩调性（暖色系=记忆/冷色系=现实/饱和=梦境）         │
│  └─ 光影关系（强光=启示/阴影=秘密/逆光=孤立）             │
│                                                           │
│  第二维：听觉层 (What You Hear)                           │
│  ├─ 环境声（风声/水声/远处的钟声/沉默本身的质感）          │
│  ├─ 身体声（呼吸/心跳/骨骼的咔嗒/衣料的摩擦）            │
│  ├─ 语言声（口音/语速/停顿/未完成的句子）                 │
│  └─ 音乐性（段落本身的节奏——长句如大提琴，短句如鼓点）     │
│                                                           │
│  第三维：心理层 (What You Feel)                           │
│  ├─ 内心声音（属性人格化系统，已设计）                     │
│  ├─ 身体感知（情绪不通过命名传达，而是通过身体反应）       │
│  ├─ 时间感知（主观时间膨胀/压缩——关键时刻变慢）           │
│  └─ 认知可靠性（哪些感知是真实的？哪些是角色的建构？）     │
│                                                           │
│  三层同时工作，由 Prompt 中的"镜头指令"统一调度             │
└─────────────────────────────────────────────────────────┘
```

**D. 新增文学风格（来自影像叙事）**

| 风格 | 灵感来源 | 语言特征 | 适用场景 |
|------|---------|---------|---------|
| **齐马蓝极简 (Zima Minimal)** | 《爱死机》齐马蓝 | 极简句式、哲学密度极高、每句话都是论点、色彩单一化、回环结构 | 人造角色/觉醒类/哲思型 |
| **潜行者凝视 (Stalker's Gaze)** | 塔可夫斯基 | 极长的环境描写、时间近乎静止、废墟中的诗意、元素（水/风/光）有人格、叙事像一条缓慢的河 | 荒野/废墟/朝圣旅程 |
| **花样年华 (Mood Indigo)** | 王家卫 | 色彩浸润一切、未说出的话是主体、时间被步调印刷拉伸、重复的意象（走廊/雨/烟）、缺席比在场更有力 | 角色关系/情感核心/遗憾与渴望 |
| **红房间梦境 (Red Room Dream)** | 大卫·林奇 | 梦境逻辑、因果断裂、诡异谷——看似正常实则不对、双重身份和重复细节、理性无法解释的恐惧 | 诅咒/疯狂/异域位面/超自然接触 |

#### 经典游戏 Lore 体系调研——叙事方法论提取

除了 D&D 本身的设定，我们还从多款世界观庞大的经典游戏中提取了叙事方法论，作为 AI 生成高质量文本的参考框架：

**① 上古卷轴 (The Elder Scrolls) — "不可靠叙述者"与多元神话**

上古卷轴的 Lore 体系被公认为电子游戏中最深邃的世界观之一。其核心设计哲学：

| 叙事技法 | 具体做法 | 我们的借鉴 |
|----------|---------|-----------|
| **不可靠叙述者** | 游戏内所有书籍、NPC 口述都是某个角色的"主观视角"，没有上帝视角的"官方真相"。使用"据说"、"传言"、"根据某某记载"等措辞 | AI 生成背景故事时，可用**多视角叙述**——同一事件从不同人物的角度有不同版本，为角色留下"真相模糊地带" |
| **嵌套宇宙论** | 从 Anu/Padomay 的原初对立 → Et'Ada（原始精灵）→ Aedra（献祭创世的神）vs Daedra（拒绝创世的神）→ 凡人世界，层层递进 | AI 世界观注入应有**宇宙论深度**——角色的信仰可以追溯到创世级别的冲突 |
| **文化相对主义** | 同一个神在不同种族文化中有不同的名字和故事（如 Akatosh/Auri-El/Alduin 实为同一神祇的不同面向） | AI 生成涉及宗教/神祇时，应根据角色种族提供**文化特定的叙述方式** |
| **游戏内书籍** | 数千本书籍构成百科全书级的世界——历史、哲学、诗歌、食谱、情色小说应有尽有 | AI 可以模拟**不同类型的文本风格**：学者论文、民间歌谣、商人日记、神庙铭文 |

关键数据源：[UESP Wiki Lore](https://en.uesp.net/wiki/Lore:Main_Page)（最权威的上古卷轴百科）| [UESP Lore Library](https://en.uesp.net/wiki/Lore:Library)（游戏内全部书籍收录）

**② 黑暗之魂/艾尔登法环 (Dark Souls / Elden Ring) — 碎片化叙事与物品描述**

宫崎英高的叙事哲学是"不告诉你故事，让你自己拼凑"：

| 叙事技法 | 具体做法 | 我们的借鉴 |
|----------|---------|-----------|
| **物品描述叙事** | 世界观不通过对话推进，而是藏在每件武器、每个戒指、每件护甲的描述文本中。一把剑的描述可能暗示一个王国的兴衰 | AI 生成装备描述时，每件物品都应承载一段微型叙事——不只是"长剑 +1"，而是"剑身上还残留着它最后一位主人的体温" |
| **环境叙事** | 场景布局本身就是故事：一具骸骨的朝向、一面破碎的旗帜、一扇从内侧封死的门 | AI 在描述角色经历的场景时，应通过**环境细节**而非直接陈述来暗示事件 |
| **留白美学** | 刻意不解释一切，留下大量未解之谜供玩家讨论 | 每个 AI 生成的背景故事应包含**至少一个未解之谜**——一段缺失的记忆、一个无法解释的伤疤、一个反复出现的梦 |
| **循环与宿命** | 故事围绕"火的循环"等宏大循环展开，个体命运与世界命运共振 | 角色的个人历史可以与世界的**宏大叙事周期**产生呼应（但不是简单的"你是天选之人"） |

**③ 巫师 (The Witcher) — 灰色道德与民间传说**

安杰伊·萨普科夫斯基笔下的世界以"没有纯粹的善恶"闻名：

| 叙事技法 | 具体做法 | 我们的借鉴 |
|----------|---------|-----------|
| **怪物志（Bestiary）** | 每种怪物不只有战斗数据，还有民间传说来源、生态习性、文化意义。知识本身是力量 | AI 生成遭遇/怪物信息时，应附带**民间传说色彩的描述**——村民的口述、老猎人的警告、学者的解剖记录 |
| **灰色道德** | 没有"正确选项"，每个选择都有代价。"血腥男爵"剧情中，施暴者同时也是受害者 | AI 生成角色的理想/羁绊/缺陷时，应避免非黑即白——**美德中藏着阴影，缺陷中包含人性** |
| **斯拉夫民间文学** | 世界观深植于东欧民间传说（午夜女妖、森林精灵、水鬼） | 提供多种**文化根基选项**——不只是西欧骑士传说，也可选择北欧、斯拉夫、中东、东亚等文化色彩 |
| **政治现实主义** | 战争不是英雄史诗，而是普通人的灾难。国王的决策影响每个村庄 | AI 在描述世界局势时，应有**自上而下的影响链**——帝国的战争 → 领主的征兵 → 角色家庭的破碎 |

关键数据源：[Witcher Wiki](https://witcher.fandom.com/)（巫师百科全书）

**④ 博德之门3 (Baldur's Gate 3) — Origin 角色系统与同伴叙事**

Larian Studios 证明了"预制角色"可以和"自定义角色"一样有深度：

| 叙事技法 | 具体做法 | 我们的借鉴 |
|----------|---------|-----------|
| **Origin 角色** | 7个预制角色各有完整背景、个人任务和世界观联系。每人都足以担当主角 | AI 可以提供**"Origin 模板"**——不只是种族+职业，而是包含个人任务钩子、NPC关系网、秘密的完整角色起点 |
| **同伴认同/反对系统** | 你的每个决策都会引起同伴的赞同或反对，揭示他们的价值观 | AI 生成角色时，可以自动生成**"与其他预设角色的潜在关系张力"**——你的信仰如何与某个典型同伴冲突？ |
| **Dark Urge 机制** | 一个"半预制"角色——外表自定义，但内心有一个预设的黑暗冲动在觉醒 | AI 可以提供**"暗面种子"**选项——在看似正常的角色背景中埋入一个逐渐觉醒的黑暗元素 |
| **背景与世界编织** | 每个同伴的个人故事都与主线剧情的不同面向深度交织（吸血鬼的主人是大Boss的部下等） | AI 生成角色时，应能将个人故事与**更大的世界冲突**有机连接 |

**⑤ 异域镇魂曲 (Planescape: Torment) — 哲学与身份认同**

被称为"游戏中的文学巅峰"：

| 叙事技法 | 具体做法 | 我们的借鉴 |
|----------|---------|-----------|
| **核心哲学问题** | 整个游戏围绕一个问题展开："什么能改变一个人的本质？" | 每个角色应有一个**核心哲学命题**——不是"打败魔王"，而是"信念能否抵抗本能？""记忆是否定义自我？" |
| **同伴即哲学立场** | 每个同伴代表一种哲学观点——唯物主义、虚无主义、信仰主义、实用主义 | AI 可以为角色生成**哲学倾向标签**，并在叙事中体现这种倾向如何影响决策 |
| **文本密度** | 80万词的对话量，文字本身就是游戏核心体验 | 我们的 AI 文本不追求"短平快"，而是提供**有密度、有层次的叙事**，用户可以选择简洁版或深度版 |
| **遗忘与记忆** | 主角失忆，过去的每一世都留下了痕迹和后果 | AI 可以生成**分层记忆**——角色"记得的"、"以为自己记得的"、"实际上被篡改的"记忆 |

- **第一层（规则层）**：确保生成内容不违反 D&D 规则——木精灵应有黑暗视觉、精灵武器训练等特质
- **第二层（世界层）**：注入世界观细节——费伦大陆的森林（高森林、克拉克森林）、精灵城市（永聚岛）、历史事件（王冠战争）

## 三、文学性文本生成体系

### 3.0 设计理念：从"填表"到"文学创作"

市面上绝大多数角色卡生成器的文本输出是**工具性的**——"你出生在一个小村庄，父母是农民，你从小对冒险充满向往"。这种文本索然无味，缺乏文学价值。

我们的目标是让 AI 生成的文本具备**文学性**——有意象、有节奏、有内心深度、有世界质感。极乐迪斯科（Disco Elysium）是我们最重要的文本参考。

### 3.0.1 极乐迪斯科的文本特色深度分析

极乐迪斯科之所以被称为"游戏文学的巅峰"，核心在于以下几个写作手法：

#### A. 技能人格化（Skills as Voices）

传统 RPG 的属性是沉默的数字。极乐迪斯科将 24 个技能变成了 24 个"内心声音"——每个技能都有独特的人格、语气和偏见：

| 技能 | 人格特征 | 语气风格 |
|------|---------|---------|
| **内陆帝国 (Inland Empire)** | 直觉、超自然感应、诗意妄想 | 梦呓般的、意象丰富的、边缘疯狂的 |
| **电化学 (Electrochemistry)** | 享乐主义、成瘾冲动 | 诱惑的、兴奋的、不计后果的 |
| **百科全书 (Encyclopedia)** | 学究、信息过载 | 掉书袋、旁征博引、时而跑题 |
| **戏剧学 (Drama)** | 浮夸表演、测谎 | 戏剧腔、夸张的、对谎言敏锐 |
| **同理心 (Empathy)** | 情感共鸣、读心 | 温柔的、感同身受的、有时过度投射 |
| **意志力 (Volition)** | 理性、自制、道德锚点 | 冷静的、鼓励的、最接近"真实自我" |
| **形体学 (Physical Instrument)** | 肌肉脑、暴力冲动 | 直接的、好斗的、崇尚力量 |
| **通灵术 (Shivers)** | 城市感应、超自然预知 | 第二人称叙述、诗意的、带寒意的 |

**核心启发**：我们的 AI 可以为角色的不同属性/技能赋予"内心声音"——力量高的角色，内心有一个粗犷直接的声音；智慧高的角色，有一个沉思的、引经据典的声音。

#### B. 思维阁（Thought Cabinet）——思想作为收集品

极乐迪斯科有 53 种"思维"（Thoughts），玩家通过与世界互动来获取。每个思维有：
- **研究阶段**：带来临时的属性惩罚（代表思考的阵痛）
- **内化阶段**：获得永久的属性加成和新对话选项
- **哲学深度**：每个思维都是一个微型哲学论文

**核心启发**：我们可以设计"角色信念系统"——角色的理想/羁绊/缺陷不只是一行文字，而是可以"研究"和"内化"的思维条目，每条附带 AI 生成的哲学化叙述。

#### C. 微反应性（Micro-Reactivity）

游戏记住并回应玩家做出的每一个细微选择——穿了什么衣服、说过什么话、做过什么梦，都会在后续文本中被引用。

**核心启发**：AI 故事生成应有"记忆"——如果用户选择了"孤儿"背景，后续所有生成的文本都应自然地回响这个设定（回忆、梦境、内心独白）。

#### D. 文学性的语言质感

极乐迪斯科的文字被评价为"可以与伍尔夫、纳博科夫、贝雷比肩"。其语言特点：
- **意象密集**：不说"你很悲伤"，而是"你的胸腔里有一片冻结的湖，每次呼吸都让冰层裂出新的纹路"
- **通感与隐喻**："黑暗尝起来像旧铜币"、"寂静有一种紫色的重量"
- **第二人称叙述**："你"直接将读者拉入角色体验
- **断裂的诗意**：句子在不该断的地方断开，制造节奏感和不安感

### 3.0.2 我们的文本风格体系

基于极乐迪斯科的启发，结合 D&D 的奇幻叙事需求，我们设计以下**可切换的文本风格矩阵**：

| 风格 | 灵感来源 | 语言特征 | 适用场景 |
|------|---------|---------|---------|
| **内心回响 (Inner Echo)** | 极乐迪斯科 | 技能作为声音、意象密集、第二人称、断裂诗意、通感隐喻 | 深度角色内心探索 |

### 3.0.4 "内心声音"系统设计

借鉴极乐迪斯科的技能人格化，我们为 D&D 的六大属性设计对应的"内心声音"，它们会在角色卡的背景故事、人格描述等位置以旁注/侧栏形式出现：

| D&D 属性 | 内心声音名称 | 人格 | 示例语气 |
|----------|------------|------|---------|
| **力量** | 铁与骨 (Iron & Bone) | 直接、好斗、保护欲强、蔑视弱者 | "你的拳头记得比你的脑子更多。每一道伤疤都是一个你活下来了的证明。" |
| **敏捷** | 影中刃 (Shadow's Edge) | 警觉、投机、优雅、不信任 | "三个出口。两个陷阱。一个机会。你在他们眨眼之前就数清了。" |
| **体质** | 不灭炉 (The Undying Furnace) | 坚忍、迟钝、乐观、顽固 | "痛？那只是你的身体在提醒你——你还活着。继续走。" |
| **智力** | 万卷阁 (The Infinite Archive) | 博学、傲慢、好奇、强迫症 | "根据《艾萨德编年史》第三卷第十七章——不，等等，你肯定没读过。" |
| **感知** | 寂静眼 (The Silent Eye) | 敏锐、多疑、悲悯、疏离 | "他在微笑，但他右手的指节发白了。他握着什么——或者在忍受什么。" |
| **魅力** | 千面镜 (The Thousand Mirrors) | 魅惑、操纵、自恋、脆弱 | "他们看着你的方式变了。不是恐惧，不是尊敬——是需要。你知道如何使用这个。" |

属性值越高的"声音"出现得越频繁、越自信；属性值低的声音偶尔出现，但会怯懦、自我怀疑。

### 3.0.5 风格 Prompt 示例

以下是"内心回响"风格的完整 Prompt 模板（最核心的风格）：

```markdown
## 风格：内心回响 (Inner Echo)

### 写作规则
- 使用第二人称叙述（"你"）
- 角色的高属性技能会作为"内心声音"插入叙述，用【声音名称】标记
- 每3-5句至少包含一个原创的通感隐喻或意象
- 句子长短交替——长句铺陈场景，短句制造冲击
- 绝不使用以下陈词滥调：
  × "从小就对冒险充满向往"
  × "命运的齿轮开始转动"
  × "一个平凡的日子"
  × "心中燃起了一团火"
- 绝不直接命名情绪（不说"你感到悲伤"），而是通过身体感受和环境来传达
- 背景故事应包含至少一个"未解之谜"——一个角色自己也不理解的记忆或感受
- 每段结尾留下一个微小的钩子，诱使读者继续

### 内心声音规则
- 仅属性值 ≥ 14 的属性会产生"自信的声音"
- 属性值 10-13 的产生"低语"（较短，不确定的语气）
- 属性值 ≤ 9 的声音罕见出现，且充满自我怀疑
- 声音以独立段落出现，格式：
  【铁与骨】"你的拳头记得。那个夜晚，火光中，你没有后退。"
- 不同声音可以互相"辩论"

### 反陈词滥调检查清单
生成后自检：
□ 有没有使用"命运/宿命"等空洞大词？→ 换成具体意象
□ 有没有直接说出情绪名称？→ 用身体反应替代
□ 开头是不是"在一个遥远的…"？→ 从一个具体的感官瞬间开始
□ 有没有"一切都变了"这种偷懒转折？→ 用具体事件替代
```

### 3.0.6 叙事深度功能设计（综合经典游戏方法论）

将上古卷轴、黑魂、巫师、BG3、异域镇魂曲的叙事精髓提炼为我们产品的具体功能：

#### 功能 N1："不可靠记忆"系统（来自上古卷轴 + 异域镇魂曲）

角色的背景故事不是"客观事实"，而是"角色自己的叙述"。AI 生成时自动包含：
- **主观叙述层**："你记得那个夜晚——至少你以为你记得"
- **他人版本层**："但你的兄弟曾说过另一个版本"
- **物证暗示层**："你腰间的那道旧伤与你的记忆不符。伤口在背面。"

用户可以选择"单一视角"（传统模式）或"多层记忆"（深度模式），后者会为同一事件生成 2-3 个互相矛盾但各自合理的版本。

#### 功能 N2："物品即叙事"装备描述引擎（来自黑暗之魂/艾尔登法环）

每件起始装备和重要物品都附带一段**微型叙事**。AI 根据角色背景自动生成：

```
传统生成：  长剑（1d8 挥砍伤害）
我们的生成：长剑（1d8 挥砍伤害）
            "剑柄上的皮革已经被另一双手磨得光亮——不是你的手。
             卖给你的商人说它来自南方。他的眼神说它来自更远的地方。"
```

#### 功能 N3："灰色人格"生成器（来自巫师 + BG3）

传统系统："理想：保护无辜者"→ 太单薄。我们的系统：

| 传统 | 我们的"灰色人格"系统 |
|------|---------------------|
| 理想：保护无辜者 | 理想：保护无辜者——**但你心里清楚，"无辜"的定义取决于你站在哪一边。上一次你保护的那个人，后来亲手烧了一个村庄。** |
| 缺陷：贪财 | 缺陷：你对金币有一种近乎虔诚的执念——**不是因为贪婪。是因为你记得口袋里空空如也的那个冬天。记得你母亲的眼神。你发誓再也不要看到那种眼神。** |
| 羁绊：保护家乡 | 羁绊：你的家乡——**一个你既渴望回去又害怕回去的地方。那里的人还记得你离开时的样子。你不确定他们会不会喜欢你现在的样子。** |

每条人格特质都有"表层"和"深层"两个维度，AI 自动生成内在矛盾和情感复杂性。

#### 功能 N4：Origin 模板系统（来自 BG3）

除了自由创建，提供一系列**深度预制 Origin 起点**，每个包含：
- 预设的核心秘密（角色初始时不知道、在冒险中逐渐揭示）
- 2-3 个 NPC 关系节点（盟友/敌人/暧昧关系）
- 一条个人任务线索（给 DM 使用的冒险钩子）
- "暗面种子"（可选的内在冲突/诅咒/预言）

示例 Origin："**最后的守誓者**"
> 你是一个已经消亡的骑士团的最后一名成员。你不知道骑士团为何覆灭——你失去了那段记忆。你只知道你的誓言还在，刺在你的前臂上，用一种你无法阅读的语言。偶尔，在深夜，那些字会轻微地发光。你不确定这是魔法还是疯狂的前兆。

#### 功能 N5：文化根基选择器（来自巫师 + 上古卷轴）

角色的文化背景不只是"种族"，还可以选择**文化色彩**，影响 AI 生成的叙事用语、神话引用和世界观细节：

| 文化根基 | 灵感来源 | 叙事特征 | 适合的角色 |
|----------|---------|---------|-----------|
| **西方古典** | 亚瑟王/圣杯传说 | 骑士精神、誓约、圣物、宫廷阴谋 | 圣骑士、贵族、骑士 |
| **北欧萨迦** | 维京/冰岛史诗 | 命运编织、荣耀之死、冰与雾、家族血仇 | 野蛮人、战士、北方背景 |
| **斯拉夫民间** | 波兰/俄罗斯传说 | 森林精灵、女巫、灰色道德、民间智慧 | 游侠、德鲁伊、术士 |
| **中东传奇** | 一千零一夜/波斯诗歌 | 沙漠学者、精灵灯、命运之轮、诗意哲思 | 法师、诗人、商人 |
| **东亚水墨** | 中国志怪/日本物语 | 山水意境、因果轮回、妖怪百鬼、师徒传承 | 武僧、德鲁伊、术士 |
| **非洲口述** | 西非/东非传统 | 智慧动物、祖灵、鼓声叙事、社区与个人 | 德鲁伊、游侠、牧师 |
| **凯尔特薄雾** | 爱尔兰/威尔士神话 | 精灵国度、时间错位、变形、音乐魔法 | 吟游诗人、精灵、德鲁伊 |

#### 功能 N6：哲学命题引擎（来自异域镇魂曲）

每个角色自动生成一个**核心哲学命题**——一个没有标准答案的问题，贯穿角色的冒险生涯：

AI 基于种族+职业+背景+人格特质的组合推导：

| 角色组合 | 生成的核心命题 |
|----------|--------------|
| 提夫林 + 术士 | "被诅咒的血脉中流淌的力量，算不算是你自己的？" |
| 矮人 + 牧师（失去信仰） | "如果你的神沉默了，你是在侍奉一个沉默的神，还是在侍奉沉默本身？" |
| 人类 + 战士（退役老兵） | "和平来了。你为之战斗的一切都实现了。那么——你现在是什么？" |
| 半精灵 + 游荡者 | "两个世界都称你为自己人。两个世界都说谎了。哪一边的谎言你更愿意相信？" |
| 龙裔 + 圣骑士 | "誓约说保护弱者。本能说点燃一切。当两者同时正确时，你听从哪一个？" |

这些命题会出现在角色卡的"核心动机"区域，并影响 AI 后续为该角色生成的所有文本。

## 四、功能规划（更新版）

### 4.1 MVP（最小可行产品）— Phase 1

> 预计开发周期：8-10 周

**模块 A：三模式角色创建入口**

| 模式 | 目标用户 | 流程 |
|------|---------|------|
| **AI 快速模式** | 新手/灵感不足者 | 通过 5-8 个对话式问答（类似DM's Vault的人格问卷，但AI驱动），自动推荐种族+职业+背景组合并生成完整角色 |
| **引导式模式** | 有经验的玩家 | 经典分步向导：物种→职业→属性→背景→装备→法术→故事，每一步有规则提示和AI建议 |
| **自由模式** | 高级玩家/DM | 完整表单编辑，所有字段可自由填写，支持 Homebrew 内容输入 |

**模块 B：完整规则引擎**

- **双版本规则支持**：用户可切换 2014 经典规则 / 2024 修订规则
- **物种系统**：SRD 全部物种 + 子种族，含种族特性自动应用
  - 2024版 10 种核心物种（含 Aasimar、Goliath、Orc）
  - 2014版 9 种经典种族 + 子种族
- **职业系统**：SRD 全部职业 + 子职业（至少到 3 级子职业选择）
  - 职业特性自动解锁
  - 法术已知/已准备数量自动计算
  - 多职业规则支持（熟练、法术槽合并计算）
- **属性值分配**：
  - 标准数组 (15, 14, 13, 12, 10, 8)
  - 点购系统 (27 点，含实时点数计算)
  - 4d6 去最低（含 3D 骰子投掷动画）
  - 2024版：属性加值来自背景而非物种
- **背景系统**：
  - 2014版：背景特性 + 技能熟练 + 工具熟练 + 语言 + 装备
  - 2024版：起源专长(Origin Feat) + 属性加值 + 技能 + 工具 + 装备
- **技能与熟练自动计算**：18项技能修正值 = 属性调整值 + 熟练加值(如适用)
- **战斗属性自动计算**：AC、先攻、速度、攻击加值、法术 DC
- **装备与负重**：起始装备 + 背景装备 + 购买，含负重计算
- **法术管理**：
  - 按职业/等级过滤可用法术
  - 戏法 + 1-9环法术列表
  - 法术槽数量追踪
  - 施法属性/DC/攻击加值自动计算

**模块 C：AI 角色故事工坊**（核心差异化模块，详见第三章"文学性文本生成体系"）

- **AI 对话式创作**：通过自然对话，AI 引导用户构建角色故事
- **"This Is Your Life" 增强版**：
  - 基于 Xanathar's Guide 的随机人生表维度（出身/家庭/成长/重大事件）
  - 骰子随机结果作为 AI 故事种子
  - AI 将零散的随机结果编织为连贯的背景叙事
- **四维人格生成**：人格特质 / 理想 / 羁绊 / 缺陷的 AI 智能建议
  - 根据选定的物种+职业+背景，推荐匹配的人格组合
  - 支持在 AI 建议基础上手动微调
- **14种文学风格可选**：内心回响 / 史诗编年 / 酒馆传说 / 暗黑低语 / 自然吟唱 / 编年档案 / 业火轮回 / 神谕碎片 / 万物有灵 / 灰烬哲人 / 齐马蓝极简 / 潜行者凝视 / 花样年华 / 红房间梦境
- **"内心声音"系统**：角色的高属性以人格化的"声音"形式出现在故事文本中（借鉴极乐迪斯科技能系统）
- **分层 RAG 知识注入**：SRD规则 + 世界观Lore + 文学风格模板 → 确保内容既合规又有深度
- **反陈词滥调引擎**：AI 自检机制，拒绝"从小对冒险充满向往"等套话
- **AI 续写与改写**：选中任意段落，请求 AI 扩写、改写或换风格
- **记忆层**：所有生成内容保持前后一致性（选了孤儿背景，后续生成自动回响此设定）

**模块 D：角色卡预览与导出**

- **实时预览**：标准三页 D&D 角色卡实时渲染
- **导出格式**：
  - PDF（标准角色卡格式 + 精美排版版本）
  - JSON（数据备份、跨平台导入）
  - PNG 角色卡片（社交分享用）
- **导入支持**：从 JSON / D&D Beyond 数据格式导入已有角色

**模块 E：本地角色管理**

- IndexedDB 持久存储（比 localStorage 更可靠）
- 角色列表：搜索、按职业/种族/等级筛选
- 角色复制、模板保存
- PWA 支持：离线可用，可安装到桌面/手机


### 5.4 关键技术决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 框架 | Next.js 14+ (App Router) | SSR/SSG 支持好，SEO 友好，API Routes 内置，边缘部署 |
| 状态管理 | Zustand + React Query | Zustand 管理本地状态；React Query 管理服务端数据缓存与同步 |
| 样式 | Tailwind + shadcn/ui | 快速开发、一致的设计系统、高度可定制 |
| 动画 | Motion.dev | WAAPI 驱动；硬件加速；弹簧物理；手势支持；Framer Motion 进化版 |
| 富文本 | Tiptap | ProseMirror 内核；可扩展插件；自定义 D&D 格式（法术/物品高亮） |
| ORM | Drizzle | 7.4KB；边缘原生；SQL-like API；超越 Prisma 的增长趋势 |
| AI 接入 | Vercel AI SDK + LangGraph | AI SDK 管流式调用；LangGraph 管 Agent 编排（内心声音/多步推理） |
| RAG | LlamaIndex + pgvector | LlamaIndex 索引 Lore 文档；pgvector 存储向量——与角色数据同库 |
| 认证 | Better Auth | 2025年新标准；插件生态；无供应商锁定 |
| 规则引擎 | RANDSUM + open5e-ts | TypeScript 原生；内置 5e 完整机制；Zod 类型安全 |
| 图像生成 | ComfyUI (Docker) | 89K stars；REST API；LoRA 风格切换；节点式工作流 |
| PDF 导出 | pdfmake | JSON 声明式——完美映射角色数据模型 |
| 测试 | Vitest + Promptfoo | 快速反馈；AI Prompt 专项测试 |
| 监控 | Langfuse | LLM 调用全链路追踪；Prompt 质量评估 |
| 缓存 | Upstash Redis | 边缘友好；Serverless；Vercel 原生集成 |
| 国际化 | next-intl | App Router 原生；ICU 格式；类型安全 |

## 九、风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|---------|
| AI API 成本 | 开源免费但 API 调用有成本 | 支持本地模型 (Ollama)；用户可 BYO API Key |
| SRD 内容有限 | 仅覆盖部分种族/职业 | 支持 Homebrew 扩展 + 接入 Open5e 额外 OGL 内容 |
| 版权/商标风险 | WotC 对 D&D 内容有保护 | 严格只使用 CC-BY-4.0 的 SRD 内容，标注来源 |
| AI 生成质量不稳定 | 故事/图像质量参差不齐 | 提供 Prompt 模板 + 用户评分反馈机制 + 重新生成 |
